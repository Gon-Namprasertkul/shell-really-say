# Dependencies

[Cargo(Rust)](https://rust-lang.org)

[Git](https://git-scm.com)

# Install

```
git clone https://github.com/ForesterBlox/shell-really-say.git
cd shell-really-say
make install
```

### To uninstall do

`make clean`
